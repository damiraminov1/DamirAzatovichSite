Бот в Telegram

УрГУПС Расписание - бот для скачивания актуального расписания. Написан на Python, с использованием PyTelegramBotAPI.
Работает 24/7 и выполняет роль агрегатора актуального расписания УрГУПС. Открыть бота в Telegram

Облачное хранилище

Облачное хранилище - сервис с веб интерфейсом для загрузки и хранения файлов. Написан на Python, с использованием веб-фреймворка CherryPy.
Получение файлов реализовано с помощью API.

Социальная сеть

Социальная сеть - проект, написанный на Python, который создавался с целью изучения веб-фреймворка Flask.
Имеет основной функционал социальных сетей (пользователи, личные страницы, подписки, лента). Реализована система регистрации пользователей, восстановления пароля.
